# Road Accident Analysis Dashboard

## Overview
This project analyzes road accident data using Power BI to identify trends, casualty rates, vehicle involvement, and high-risk locations.

## Tools Used
- Power BI
- Excel
- Power Query
- DAX
- SQL

## Key Features
- Total Accidents KPI
- Casualty Analysis
- Vehicle Type Analysis
- Road Type Analysis
- Monthly Trend Analysis
- Interactive Filters and Slicers

## Project Outcome
The dashboard provides meaningful insights into accident patterns and helps support data-driven decision making.
